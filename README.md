# PrismaORMv7:
- Projeto de Teste com Prisma ORM (versão atualizada)

## Tecnologias utilizadas

- Node.js
- TypeScript
- Prisma ORM (nova versão)
- PostgreSQL
- TSX (para rodar o TypeScript sem compilar)

## Como rodar o projeto
```
1️⃣ Instalar dependências
npm install
2️⃣ Gerar o Prisma Client
npx prisma generate
3️⃣ Rodar migrações
npx prisma migrate dev
4️⃣ Executar o projeto
npx tsx index.ts
```
